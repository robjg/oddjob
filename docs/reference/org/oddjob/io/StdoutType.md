the console.

| Title | Description |

### Examples
#### Example 1 <a name="example1"></a>

Copy from a file to stdout.


```xml
<copy>
    <input>
        <file file="${my.file}"/>
    </input>
    <output>
        <stdout/>
    </output>
</copy>

```
